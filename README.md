PERANAN DATA DALAM PENGAMBILAN KEPUTUSAN BISNIS
Tujuan Praktik
Praktik ini bertujuan untuk memahami dasar-dasar analisis data menggunakan bahasa pemrograman Python dengan memanfaatkan library pandas. Dataset yang digunakan adalah dataset Titanic yang memuat informasi penumpang kapal RMS Titanic.

 Secara khusus, tujuan dari praktik ini adalah:
Memastikan lingkungan pemrograman berjalan dengan baik
Mahasiswa mampu menjalankan kode Python di Google Colab tanpa error.
Memahami cara mengimpor dan menggunakan library pandas
Mahasiswa dapat menggunakan pandas untuk membaca dan mengelola data dalam bentuk DataFrame.
Memuat dataset dari sumber online (GitHub)
Mahasiswa mampu mengambil dataset langsung dari repository GitHub menggunakan URL dan membacanya dengan pd.read_csv().

  Mengenali struktur dataset
Mahasiswa dapat:

 Menampilkan data awal menggunakan df.head()
Melihat jumlah baris dan kolom
Mengidentifikasi tipe data (int64, float64, object)
Mengetahui adanya missing values melalui df.info()
Melakukan analisis data sederhana
Mahasiswa mampu:
Menghitung jumlah data berdasarkan kategori tertentu (contoh: jenis kelamin dengan value_counts())
Menghitung rata-rata numerik (contoh: rata-rata usia dengan mean())
Memahami pentingnya data cleaning sebelum analisis lanjutanMahasiswa menyadari bahwa adanya data kosong (missing values) memerlukan proses pembersihan agar hasil analisis lebih akurat.

 Kesimpulan Tujuan
Melalui praktik ini, mahasiswa diharapkan memiliki pemahaman dasar mengenai proses analisis data, mulai dari membaca dataset, memahami struktur data, hingga melakukan analisis statistik sederhana. Praktik ini menjadi fondasi awal sebelum mempelajari teknik analisis data yang lebih kompleks seperti visualisasi data, pengolahan data lanjutan, maupun machine learning.
